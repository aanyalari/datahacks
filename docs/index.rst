CCE Mooring — project documentation
====================================

This documentation supports a **dual-track** hackathon project:

- **Data Analytics:** exploratory analysis, data quality, and interpretable visuals (see Marimo + Streamlit).
- **Machine Learning / AI:** time-ordered supervised forecasting with an explicit persistence baseline.

Primary dataset
---------------

`California Current Ecosystem (CCE) Mooring Array <https://mooring.ucsd.edu/cce/>`_ (Scripps / SIO collaborators).
